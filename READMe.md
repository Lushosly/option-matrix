# Quant-3D: Option Pricing Enging

An interactive Financial Engineering tool that visualizes the Black-Scholes-Merton model in 3D space, to translate complex derivative princing
formulas (Partial Differential Equations) into interactive, GPU-accelerated visualizations using Python.

---

## Key Features
- Real time calculation: Solves the Black-Scholes equation instantly for Call/Put options.
- Greeks Engine: Calculates first and second-order sensitivities (Delta, Gamma, Vega, Theta, Rho).
- 3D Volatility Surface: Interactive Plotly mesh rendering showing how option price correlates with spot price and implied volatility simultaneously.

---

## Tech Stack
- Language: Python 3.10+
- Math: Numpy, SciPy (Statistical Norm Functions)
- Viz: Plotly 3D (WebGL)
- Framework: Steamlit
